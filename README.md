# Wizeline_linkedln
Wizeline Linkedln challenge

## Requirements

* node Js
* testcafe
* testcafe-reporter-html

```bash
npm install -g testcafe
npm install -g testcafe-reporter-html
```

## Usage

Chrome

```bash
 npm run test-all-chrome
```

Firefox

```bash
 npm run test-all-firefox
```

Edge

```bash
 npm run test-all-edge
```
